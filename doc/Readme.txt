===============================================================================

　HTTDir

===============================================================================


■ソフトの概要

　起動するだけで簡単に使えるウェブサーバーです。
　ドキュメントルート（公開フォルダ）の配下にあるファイルの閲覧・ダウンロードのみ可能です。


■動作環境

　Windows 10 Pro または Windows 10 Home


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■起動方法

　HTTDir.exe を実行して下さい。


■使い方

　起動するとタスクトレイ上にアイコン（緑と赤の正方形が重なった絵柄）が表示されます。
　この時点でサーバーは稼働しているので、http://localhost/ を開くとサンプルページが表示されるはずです。
　ドキュメントルートは、DocRoot フォルダに設定されています。

　ポート番号・ドキュメントルートは「設定」から変更出来ます。
　設定を開くには、アイコンを右クリックして「設定」を選択して下さい。


■終了方法

　タスクトレイ上のアイコンを右クリックして「終了」を選択して下さい。


■補足

　同じＩＰアドレスからの最大同時接続を２０に制限しています。

　- - -

　本サーバーは送受信データを一時ファイルに書き出します。
　仮に不正なリクエストがあった場合、不正な受信データを書き出した一時ファイルがウィルスと見なされ
　ウィルス対策ソフト等によって隔離等の処置が行われる可能性があります。

　対策など追加情報
　https://github.com/stackprobe/HTT/blob/master/HTT_memo/HTT_e0001.txt
